# Directory Structure

```
packages/
  */  - contains the component code
    src/
      *.tsx - the component code
      *.styles.ts - the component styles
      *.stories.tsx - the component stories
      *.tests.tsx - the component tests

storybook/

playground/
  src/
    *.tsx - the application code
    *.styles.ts - the application styles

website/
  src/
    *.mdx - the documentation files (mdx is a variation of markdown)
    *.tsx - the website code
    *.styles.ts - the website styles

```
