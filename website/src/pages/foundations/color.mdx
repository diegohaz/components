`salad-ui` uses a themable set of color variables in order to cater to support an ecosystem of plugins e.g. Yoast, Woo, etc.

## Background and Surface colors

The `background` color is used for the background color of the page. Any text on a `background` color must use one of the `onBackground` colors.

The `surface` color is usually used to form some kind of a grouping on top of a `background` color. Any text on a `surface` color must use one of the `onSurface` colors.

<BackgroundAndSurface />

## Primary and Secondary colors

The `primary` color is used for common interface elements like the navigation and inputs. Any text on a `primary` color must use the `onPrimary` color.

The `secondary` color is used for less common interface elements like buttons. Any text on a `secondary` color must use the `onSecondary` color.

<PrimaryAndSecondary />

## Success, Warning and Error colors

The `success`, `warning` and `error` colors are used sparingly to indicate a success, warning or error state (note: color should not be the only indicator of a success, warning or error state). Any text on a `success`, `warning` or `error` color must use the applicable `onSuccess`, `onWarning`, or `onError` color.

<SuccessWarningAndError />

## Choosing colors for a custom theme

Ensure `on<color>` is accessible. See the guide for [Customising the theme](../../guides/customising-the-theme).

## Installation

```bash
npm install @salad-ui/color
```

## Usage

`salad-ui` provides a number of mixins for using the color variables. Each mixin can take a single value applied at all breakpoints or a map of values at the desired breakpoint.

```jsx inline=false
import styled from 'styled-components';
import {color, borderColor, backgroundColor} from '@salad-ui/color';

const Alert = styled.div`
  ${color('error')}
  ${borderColor('error')}
  ${backgroundColor({mobile: 'surface', desktop: 'backgorund'})}
  border-radius: 3px;
  border-style: solid;
  border-width: 2px;
  padding: 1em;
`;

render(<Alert>Danger, Will Robinson!</Alert>);
```

- `color` - applies color to the text
- `backgroundColor` - applies color to the background
- `borderColor` - applies color to the border
- `borderColorTop` - applies color to the top border
- `borderColorRight` - applies color to the right border
- `borderColorBottom` - applies color to the bottom border
- `borderColorLeft` - applies color to the left border
