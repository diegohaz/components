A basic typography component.

## Installation

```bash
npm install @salad-ui/typography
```

## Usage

```jsx
import {Title} from '@salad-ui/typography';

<>
  <Title size="large">The quick brown fox jumps over the lazy dog</Title>
  <Title size="medium">The quick brown fox jumps over the lazy dog</Title>
  <Title size="small">The quick brown fox jumps over the lazy dog</Title>
</>;
```
