A basic typography component.

## Installation

```bash
npm install @salad-ui/typography
```

## Usage

```jsx
import {Body} from '@salad-ui/typography';

<>
  <Body>The quick brown fox jumps over the lazy dog</Body>
  <Body isSmall color="error.main">
    The quick brown fox jumps over the lazy dog
  </Body>
</>;
```
