## Container

The container constraints its content to a maximum width of `1200px`, centering it horizontally when the screen is larger.

### Installation

```bash
npm install @salad-ui/container
```

### Usage

```jsx inline=false
import styled from 'styled-components';
import {container} from '@salad-ui/container';

const Content = styled.main`
  ${container}
  background-color: aliceblue;
`;

render(<Content>Content goes here</Content>);
```

> There is also a [`Container` component](/components/container).
